# Global Sales & Profit Performance Dashboard (Tableau)

## Project Overview
This project showcases an executive-level Tableau dashboard built using the Global Superstore dataset to analyze global sales performance, profitability, discount behavior, and order trends. The dashboard is designed to support data-driven business decisions through clear KPI definitions and interactive visualizations.

## Business Objectives
- Define and track core commercial KPIs (Sales, Profit, Discount, Orders)
- Analyze sales versus profit efficiency across regions and categories
- Identify high-performing markets, products, and customer segments
- Evaluate order growth trends to support strategic planning
- Deliver clear, executive-ready insights

## Tools & Technologies
- Tableau Desktop  
- Excel / CSV (Global Superstore Dataset)

## Dashboard Features
- Executive KPI summary including total sales, total profit, and average discount
- Global market analysis using country-level sales mapping
- Sales vs profit comparison by region
- Year-over-year order trend analysis
- Segment performance across Consumer, Corporate, and Home Office
- Category-level sales and profit contribution analysis
- Customer profitability analysis using sales vs profit visualization
- Top-performing cities and products

## Key Insights
- Technology is the highest profit-contributing category
- Certain regions show high sales but lower margins, indicating discount or cost inefficiencies
- A small number of products and cities drive a large share of total revenue
- Order volumes show consistent growth over time
